Để chạy demo được cần:

1. Thiết lập SQL Server, từ file "/CT239/Controller/DAO.py" đổi đường dẫn connect thành địa chỉ của máy người demo.
Sau đó tạo database và table với các trường giống như trong ảnh SQL.png đính kèm theo

#####################################################################################################################
2. Cài đặt python: https://www.python.org/downloads/

Cài đặt pip : https://pip.pypa.io/en/stable/installation/


#####################################################################################################################
3. cài đặt Extension: Python


#####################################################################################################################
4. Cài đặt thư viện:
Từ terminal gõ những lệnh phía dưới để cài đặt thư viện hỗ trợ cho việc demo ứng dụng.

Thư viện dùng để vẽ đồ thị "matplotlib": pip install matplotlib

Thư viện hỗ trợ kết nối với nguồn dữ liệu ODBC từ Python "pyodbc": pip install pyodbc  

Thư viện hỗ trợ xử lý dữ liệu "pandas": pip install pandas

Thư viện hỗ trợ làm việc với ma trận và mảng "numpy": pip install numpy
